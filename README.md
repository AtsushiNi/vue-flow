# Vue-Flow サンプルアプリ

Vue-Flowを使用したインタラクティブなER図のサンプルアプリケーションです。

## 主な機能
- インタラクティブにテーブルを表示
- テーブル間のリレーションシップを可視化
- サイドバーで表示するテーブルを選択
- レイアウトの自動調整
- クリックで関連するテーブルを表示・ハイライト

## インストール方法

```bash
pnpm install
pnpm run dev
```
